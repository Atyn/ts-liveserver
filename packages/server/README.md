## Start server

### NPX

```bash
npx @ts-liveserver/server
```

### Installed

```bash
npm install @ts-liveserver/server
ts-liveserver
```

### Cli options

#### Serve from directory

```bash
npx @ts-liveserver/server ./aDirectory/to/serve/from
```

#### Watch

```bash
npx @ts-liveserver/server --watch .
```
